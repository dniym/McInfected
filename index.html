This will hopefully help you figure out how to use Infected as fully as possible! Infected was a plugin I started making not long after MW3 introduced Infected, originally It was meant to be a "How much java do I actually know" but slowly it turned into an actual project.

What is "Infected"?

Infected is a popular game mode from Call of Duty's Modern Warfare 3. This plugin tries its' best to replicate that gamemode and turn it into a Minecraft minigame. The objective of this game for humans is to simply outlast and fight the zombies. But you only have one life. Once you get bitten, then you turn into a zombie and are forced to infect the ones who were once your allies and friends. Humans. If you happen to get caught by the horde, your objective as a zombie is to infect all the humans before the timer is up.

Features

  • Saves inventory, armor, exp, levels and the player's last known spot
  • Auto-starts when the required amount of players join the lobby
  • Ability to vote for the map you want to play
  • Multiple spawns for arenas
  • Score & points system
  • Shop - spend your points to buy new items
  • Grenades and Classes
  • Non Spout GUI for menus
  • DisguiseCraft support
  • iDisguise Support
  • LibsDisguise support
  • Vault Support
  • Factions Support
  • mcMMO Support
  • CrackShot Support
  • Uses the new scoreboard feature
  • Tracks kills/deaths/killstreaks and playing time
  • Rewards for winning(Last human alive)
  • Grenades (can be customized to your liking)
  • Define classes for players to use
  • Use your economy instead of points
  • Signs can be used for commands/showing game info
  • Optional addons

Addons

Infected does have the ability for addons to be created by devs, and to start it off i have created two. The first addon is Ranks, this addon allows for ranks to be added to Infected, and each rank can have default classes, and additional permissons granted to the player well in game. The second addon is for anyone who has a Dedicated Infected Server, in other words it has a force join, no longer needs /Infected before commands, and prevents leaving, perfect for a server devoted to only Infected.

If you'd like to enhance your gameplay experience, there are currently a few plugin addons available for Infected.

Disguises

The first few are for Disguises, but for those you need plugins such as DisguiseCraft, LibsDisguises or iDisguise. These plugins have the ability to disguise infected players as actual zombies! If you're using disguises please be sure to install the latest and greatest ProtocolLib. To use Disguise Support Just enable it in the config

Vault

Next would be Vault. With this you can get Infected to use your economy. Just enable it in the config for points to be given as cash, if you want to use vault for shops, just put a '$' in front of the price.

mcMMO

And we all know how awesome mcMMO is, but it can get annoying when playing a pvp minigame, so I have included it as an addon, and with this as an addon, it'll ignore alll the bonus damage, thats right, no more hitting big well other guys are hitting small, this way the pvp is 100% fair

Factions

Factions was also included in this, A lot people have already experienced how awesome Factions is, so to help it out, i have added Factions support, this will allow allies and fellow faction members to attack each other regardless of where the game is(Wilderness, Claimed Land, Wilderness)

CrackShot

And finally we have CrackShot, now this is an awesome plugin that allows for guns without needing spout. If you want to give guns in classes, or allow players to buy them through a shop, check out this page in the CrackShots wiki. Support
